NAME

capricornd - the capricorn server

SYNOPSIS

capricornd [options] install
capricornd [options] tail
capricornd [options] start
capricornd [options] stop
capricornd [options] restart
capricornd [options] reboot
capricornd [options] ping
capricornd [options] console
capricornd [options] attach

OPTIONS

These options apply to all the commands:

-r=path, --root=path
Path to the root of the node. This is the directory conatining the etc and var directories.

These options apply to start and console:

-p=number, --port=number

The port number to use for the Erlang distribution. Usually you don't need this.

--cert=path

The path to the nodes SSL certificate. (defaults to: etc/capricorn/cert.pem)

--ca-cert=path

The path to the CA's SSL certificate. (defaults to: etc/capricorn/ca.pem)

COMMANDS

install
Install the config files.

tail
Tail the log files.

start
Start a node.

stop
Stop a node.

restart
Restart a node.

reboot
Reboot a node.

ping
Ping a node.

console
Start a node and open an erlang shell.

attach
Attach to a node and open an erlang shell.
