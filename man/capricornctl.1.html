OPTIONS

These options apply to all the commands:

-c=cluster, --cluster=cluster

Send the command to cluster. If cluster is not set capricornctl will look for a default cluster.

-e=environment, --env=environment, --environment=environment

When environment is set capricornctl will look for a .capricorn.yml file (see capricorn.yml(5)) and read the configuration from the environment specified in there. The options provided by the environment can be overwritten by specifying the as options.

These options apply to deploy, apps:*, machines:reboot, machines:selfupdate:

-m=machine, -n=machine, --machine=machine, --node=machine
Send the command to machine. If machine is not set capricornctl will look for a default machine.

These options apply to deploy, apps:show, apps:init, apps:start, apps:restart, apps:relink, apps:stop, apps:fupdate, apps:domains:*, apps:gems:*:

-a=application, --app=application, --application=application
Send the command to application. If application is not set capricornctl will look for a default application.

COMMANDS

Commands for managing applications:

deploy
Deploy the last build gem to the current application.

apps:list
List all the applications on a machine.

apps:show
Show all information about an application.

apps:create name environment domains...
Create a new application on machine.

  • name is the human name of the application.
  • environment is the Rails environment of the application and can be development, test, staging, production.
  • domains is a list of domains to bind to this application.

apps:import name environment domains...
Import an existing application on machine.

  • name is the human name of the application.
  • environment is the Rails environment of the application and can be development, test, staging, production.
  • domains is a list of domains to bind to this application.

apps:init
Create the .capricorn.yml file (see capricorn.yml(5)).

apps:start
Start a stopped application.

apps:restart
Restart an application. this will touch tmp/restart.txt.

apps:relink
Relink an application. this will touch tmp/restart.txt and tmp/relink.txt.

apps:stop
Stop a stopped application.

apps:fupdate
Force update an application.

apps:domains:list
List all the bound domains for this application.

apps:domains:add domain
Bind a new domain to this application.

apps:domains:remove domain
Unbind a domain from this application.

apps:gems:list
List all the used gems for this application.

apps:gems:add gem
Add a new gem to this application.

apps:gems:remove gem
Remove a gem from this application.
